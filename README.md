# WTRMLN CHAT

> Simple, **_anonymous_ chat rooms** that persist until a user ***spits*** the seed and deletes the room.

## HOW TO USE WTRMLN CHAT:

- Enter your username.
- Set a seed to create or join.
- Join the room.
- Chat!
- Spit the seed to delete the chat room forever.
